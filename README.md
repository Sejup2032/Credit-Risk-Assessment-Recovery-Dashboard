# Credit-Risk-Assessment-Recovery-Dashboard

## 🔍 Project Overview
This project focuses on identifying high-risk borrowers and improving financial recovery tracking.  
I created a **credit risk monitoring system** that merges customer, loan, and transaction data from multiple sources, runs a predictive model to classify risk, and visualizes insights in Power BI and Dash.
The goal was to make risk analysis faster, transparent, and more proactive for financial institutions.

---

##  What I Built
- **Data Pipeline:** Used Azure Data Factory to consolidate loan and repayment data from multiple databases.  
- **Feature Engineering:** Cleaned and merged datasets using Python, focusing on behavior patterns and repayment trends.  
- **Modeling:** Built a Random Forest model to flag risky borrowers 
- **Dashboard:** Designed an interactive Power BI and Dash dashboard showing delinquency trends, exposure, and regional recovery rates.  
- **Automation:** Set up alerts for anomalies and scheduled data refreshes.  

---

## 📈 Key Outcomes
- Increased credit risk detection accuracy  
- Automated reporting saved time 
- Improved transparency in loan recovery and compliance tracking
